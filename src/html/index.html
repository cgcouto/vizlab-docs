VizLab user documentation

This is a repository containing user documentation for the VizLab at Carnegie Observatories!

I really hope these resources are helpful to you. If you have any questions, the best way to reach me is by submitting an issue on this repository. Do that and I’ll help you out to the best of my abilities!

-Chris Couto, Schmidt Scholar working on the VizLab (cgcouto on github)

You’re looking for the quick-start section!

The troubleshooting guide should be helpful!
