Software index

This section collects resources about all the specialized software that has been built for the VizLab. The ‘data visualization’ section touches on the VizLab’s custom data visualization software and includes a user guide along with a series of tutorials for loading various scientific datasets. We also have a few pre-built scenes that have been constructed for scientific outreach - for each of those you can find important details for running and interacting with the scenes, along with some relevant scientific information.
