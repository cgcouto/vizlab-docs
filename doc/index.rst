VizLab user documentation
=========================

This is a repository containing user documentation for the VizLab at Carnegie Observatories! 

I really hope these resources are helpful to you. If you have any questions, the best way to reach me is by submitting an issue on this repository. Do that and I'll help you out to the best of my abilities!

-Chris Couto, Schmidt Scholar working on the VizLab (cgcouto on github)

FAQ
^^^
   I need to get the VizLab up and running!

You're looking for the :doc:`quick-start </about-software/QUICKSTART>` section! 

   I'm having problems with the VizLab!

The :doc:`troubleshooting guide </about-hardware/TROUBLESHOOTING>` should be helpful!

   I want to know more about how the VizLab's hardware/software actually works!

Check out the sections on :doc:`hardware </about-hardware/README>` and :doc:`software! </about-software/README>`

   I want to use the VizLab's custom data visualization software!

We have a specific page for that :doc:`here! </about-software/USERGUIDE>` You can also check out the documentation - both user and developer - in the codebase repositories.
